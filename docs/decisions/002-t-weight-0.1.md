# ADR-002: T 가중치 0.2 → 0.1 축소

**날짜**: 2026-04-19
**상태**: 채택

## 배경

초기 C 0.4 / P 0.4 / T 0.2. T는 이촌역(사용자 거주지) 기준 자차 통근시간. 운영 2주 후 sensitivity 분석: T가 Top30 중 16개 동을 좌우. 본인 편의 지표일 뿐 입지 품질과 무관한데 과대 영향.

## 결정

- **C 0.45 / P 0.45 / T 0.1**로 재조정.
- PWA 메인에 가중치 슬라이더 추가 → 사용자가 런타임 재가중.

## 대안

- **T 완전 제거**: 먼 소도시까지 Top30 진입 (답사 불가). guardrail 필요 → 0.1.
- **자차 대신 대중교통**: ODSay. 사용자 실이동 수단이 자차라 Kakao Mobility 주력, ODSay는 display.

## 결과

- Top30 구성 안정화, 저평가됐던 서울 belt (노원 등) 재진입.
- 슬라이더 0~0.3 조정해도 전체 질서 유지 확인.
