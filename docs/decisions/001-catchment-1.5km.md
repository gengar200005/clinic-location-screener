# ADR-001: catchment 1.5km 기반 P_raw + C density 분모 통일

**날짜**: 2026-04-20
**상태**: 채택

## 배경

행정동 경계의 인위성. "작은 동 + 인접 동 대단지" 상황에서 작은 동의 P_raw가 과소평가됨 (행신2동 사례). 초기 P_raw는 행정동 내 40+ 인구만, C density 분모도 마찬가지.

## 결정

- **P_raw** = catchment_pop_1.5km × ratio_40plus (중심점 반경 1.5km 배후 인구 중 40+).
- **C density 분모** = catchment_pop_40plus (동일 반경).
- 중심점은 WorldPop 100m 격자 인구 가중 좌표.
- 반경 1.5km는 P·C density 모두 통일 (도보 + 단거리 자차 통근권).

## 대안

- **격자(D안) 500m**: 기각. 답사 대상이 행정동 단위 매물/상권 → 정밀도 대비 실용성 낮음.
- **500m 반경 유지**: 도심 P 인플레 + Top30 동률 68개. 1.5km로 13개로 해소.
- **ratio_40plus 생략**: 소화기내과 실 환자풀과 괴리 커 기각.

## 결과

- Top30 sido: 서울 22 / 경기 8 (노원·강북·도봉 고령 belt 부상).
- 행신2동 사례 정상화.
- Top50 export에서 t_raw > 50분 컷.
